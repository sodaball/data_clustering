### 具体内容：

#### 1. 对iris数据集进行聚类分析

使用sklearn的库对iris数据集进行读取和聚类分析，分析出iris数据集中的类别

算法步骤：

1. 随机选择K个点作为初始的聚类中心
2. 对于数据集中的每个点，计算其到每个聚类中心的距离，并将其分配到最近的聚类中心
3. 更新每个聚类的中心点，使其成为该聚类所有点的平均值
4. 重复步骤2和3，直到聚类中心不再发生变化或达到预设的最大迭代次数
